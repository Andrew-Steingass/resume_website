User Guide by Andy Steingass
A quick guide to important features.

If you want to reorder the headers and want the side bar quick draw items to be in the same order, you will have to arrange hugo.toml in the same orientation as your markdowns.
Adding to "themes\resume\layouts\partials\nav.html" will allow you to add extra items to the side bar.
"themes\resume\static\css\resume.css" will allow you to adjust the spacing and formatting to an extent.
"content\_index.md" is the main markdown for this website. adjust it to update resumes.
If you want to re arrange the header, you will most likely need to re-arrange in here themes\resume\layouts\partials\about.html
"themes\resume\i18n\en.json" will allow you to adjust side bar options
